# Data-Cleaning-using-MS-SQL
# Project Summary
This project demonstrates how to clean and prepare a COVID-19 dataset using Microsoft SQL Server. It focuses on transforming raw data into a structured, analysis-ready format through SQL queries and logic.

 # Tools Used
- Microsoft SQL Server
- SSMS (SQL Server Management Studio)
- GitHub for version control
# Cleaning Tasks Covered
-  Removed duplicate records
-  Handled NULL values and missing data
-  Standardized date formats
-  Renamed columns for clarity
-  Filtered out irrelevant rows (e.g., test entries or incomplete records)
 # Files Included
- CovidProject.sql → SQL script with all cleaning queries
- README.md → Project overview and documentation
# Why It Matters
Clean data is the foundation of reliable analysis. This project shows how a data analyst can:
- Use SQL to clean and structure public health data
- Prepare datasets for exploration, visualization, or reporting
- Apply best practices in query design and data hygiene
